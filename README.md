# Salairia

> Votre rémunération, sans zone d'ombre.

Site SEO français sur la rémunération (salaire, freelance, portage, auto-entrepreneur, SASU). Simulateurs gratuits, guides clairs, comparatifs neutres.

## Stack

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · Vercel

## Développement

```bash
npm run dev      # dev server (http://localhost:3000)
npm run build    # production build
npm run start    # run production build
npm run lint     # eslint
```

Voir [`CLAUDE.md`](./CLAUDE.md) pour l'architecture, la charte graphique et les décisions techniques verrouillées.
